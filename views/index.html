URL Shortener Microservice

Hello!

Add a web address to the end of the existing web address in the address bar to get a short url and an original url, both redirecting you to the original url.

An example of this might be the following:

https://url-shortener123.glitch.me/https://www.google.com,

The output will look something like this:

{short_url: "https://url-shortener123.glitch.me/new", original_url: "https://www.google.com"}
